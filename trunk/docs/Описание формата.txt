1я строчка - колличество кадров (N_Frames)
2я строчка - колличество точек в шейпе (N_Points)
далее следует N_Frames следующих блоков, описывающих по одному кадру:
{
1я строчка блока - время длительности кадра (dt)

N_Points строчек - точки границы - 2 дробнодесятичных числа, разделенных пробелом. (например 158.0691 202.7653)

(N_Points + 2) строчка содержит либо слово "Motion" либо слово "Passive". Motion обозначает, что для данного кадра заданны скорости на клапане. Passive означает что скорости на клапане не заданны.

(N_Points + 3) строчка (присутствует только если предыдущий параметр был "Motion") хранит вектор скорости на клапане (2 дробнодесятичных числа через пробел). Если предыдущая строчка содержит слово "passive", то этой строки вовсе нет.
}
